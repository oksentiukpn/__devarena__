Інфраструктура:

Повністю докеризоване середовище. Nginx як reverse-proxy (SSL, захист, Rate Limiting). Автоматизований CI/CD pipeline через GitHub Actions із розгортанням на VPS при пуші в main. Інтеграції з Google OAuth та Resend API.
